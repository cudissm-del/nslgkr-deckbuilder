# 도감 녹화 가이드 (최소 수고 데이터 수집)

목표: 무장·전법 도감을 **영상으로 한 번에** 담으면, 프레임 추출→중복 제거→읽기→JSON 구조화는 전부 Claude가 처리.

## 1. 무엇을 녹화하나 (우선순위)
1. **전법 도감(목록)** — 전법 이름 전체. *덱의 핵심.*
2. **무장 도감(목록)** — 장수 이름·세력·병종 전체.
3. (여유되면) **전법 상세를 하나씩 열며** 녹화 — 효과문·발동률까지. 목록보다 오래 걸리니 나중.

## 2. 녹화 방법 (Windows)
- **Xbox Game Bar**: 게임 창을 클릭해 활성화 → `Win + Alt + R` 로 녹화 시작/종료.
  저장 위치 기본값: `영상\Captures\`.
- 또는 이미 쓰시는 **OBS / Stream Deck / 팟플레이어 녹화** 아무거나 OK.
- 해상도는 지금 창 크기 그대로면 충분(방금 올려주신 풀샷 수준이면 아주 잘 읽힘).

## 3. 스크롤 요령 (정확도 ↑)
- **천천히**. 한 페이지(또는 몇 줄) 보이면 **1초 정도 멈췄다가** 다음으로.
- 관성 스크롤로 휙 넘기지 말 것 — 흐릿한 프레임은 못 읽어요.
- 도감 맨 위 → 맨 아래까지 빠짐없이. 필터(세력별 등) 있으면 탭마다 한 번씩.

## 4. 영상 넣는 곳
- 녹화 파일을 이 폴더에 복사: `E:\NSLGKR\deckbuilder\captures\videos\`
  - 예: `videos\전법도감.mp4`, `videos\무장도감.mp4`
- 넣고 나서 채팅에 "영상 넣었어"라고 알려주세요.

## 5. 그다음 (Claude가 함)
1. `tools/extract_frames.py` 로 프레임 추출 + 중복 제거 → `captures/frames/<영상이름>/`
2. 선명한 대표 프레임만 읽어 `data/heroes.json`·`data/tactics.json` 에 누적(중복 자동 병합).
3. 빠진 값(효과문·스탯 등)은 목록 확보 후 우선순위로 상세 보완.
