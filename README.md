# 삼국지: 천하결전 — 덱빌더 (nslgkr-deckbuilder)

게임 **삼국지: 천하결전**의 전투 메커니즘을 데이터화하고, 좋은 덱(부대·장수·전법 조합)을 설계하기 위한 프로젝트.

- **지식 베이스**: [`PROJECT.md`](PROJECT.md) — 전투 구조·확정 메커니즘·데이터 현황·다음 과제
- **데이터 스키마**: [`SCHEMA.md`](SCHEMA.md)
- **분석 문서**: [`docs/`](docs/)
- **데이터베이스**: [`data/`](data/) — `arms` · `heroes` · `tactics` · `combat` · `glossary`
- **덱빌더 사이트**: `docs/`(또는 `/site`) — GitHub Pages 예정

## 데이터 파일

| 파일 | 내용 |
|---|---|
| `data/arms.json` | 병종 4종 + 상성 순환 (방패>궁>창>기마>방패) |
| `data/heroes.json` | 무장 67명 (스탯·성장치·전법·역할·병종) |
| `data/tactics.json` | 전법 61종 (유형·특성·발동률·효과) |
| `data/combat.json` | 전투 계산 파라미터 (편성보너스·공급/병력 페널티·상태이상·버프) |
| `data/glossary.json` | 게임 용어 사전 80+ |

## 데이터 수집 파이프라인

도감/상세는 **화면 녹화 → 프레임 추출(`tools/extract_frames.py`) → 판독**, 그림 많은 공략글은 **PDF 저장 → 페이지 렌더 → 판독**. 원본(영상·프레임)은 용량 문제로 저장소에서 제외(`.gitignore`).

> 데이터 출처: 게임 내 도감·상세 화면, 공식 용어 가이드 및 전투 교과서(네이버 카페). 수치는 시즌/패치에 따라 변경될 수 있음.
